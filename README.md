## 🚀 CARA MAIN

1. **Buka `index.html`** di browser
2. **Enter gamertag** (nama kamu)
3. **Pilih character** (klik salah satu dari 4 class)
4. **Klik "DROP INTO BATTLEFIELD"**
5. **Move dengan WASD**, aim dengan mouse!

## 🎮 CONTROLS

### Keyboard
- **W A S D** - Move (WASD movement)
